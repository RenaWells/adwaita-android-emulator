Third-Party Licenses

This page credits dependencies, runtimes, services, and tools used by the app.

This is an attribution and license notice page for third-party components. It is not a license grant for this project's own source code.

Included in the AppImage

The AppImage bundles the launcher and AppImage runtime, not a full GTK or Android SDK stack.

  • AppImage type 2 runtime - MIT license. Source: AppImage/type2-runtime. Used as the executable runtime prepended to the AppImage payload.

Required Host Runtime

Installed by the operating system or the user, not bundled into the AppImage.

  • Python - Python Software Foundation License. Source: python.org. Used as the application runtime.
  • GTK 4 - LGPL-2.1-or-later. Source: gtk.org. Used for application widgets through PyGObject.
  • libadwaita - LGPL-2.1-or-later. Source: GNOME libadwaita. Used for Adwaita widgets and styling.
  • PyGObject / GObject Introspection - LGPL-2.1-or-later. Source: pygobject.gnome.org. Used for Python bindings to GTK, libadwaita, GLib, Gio, and related libraries.

Downloaded by Setup

These are fetched or installed by the setup wizard or script on the user's machine.

  • Android SDK tools, emulator, platform-tools, and system images - Android SDK License Agreement and component-specific licenses. Source: Android Studio downloads. Used for SDK management, ADB, emulator binaries, and Android system images.
  • Arch Linux bootstrap and packages - per-package licenses from Arch Linux and upstream projects. Source: archlinux.org. Used as the optional glibc runtime path on Alpine/musl systems.
  • OpenJDK and distro packages - per-package licenses. Installed as host dependencies for SDK tooling and runtime integration.

Build and Hosting Tools

Used to build, publish, host, or operate the project.

  • squashfs-tools / mksquashfs - GPL-2.0-or-later. Source: squashfs-tools. Used to build the AppImage SquashFS payload.
  • libarchive / bsdtar - BSD-style license. Source: libarchive.org. Used for archive extraction during setup.
  • GitHub CLI - MIT license. Source: cli/cli. Used for publishing and repository workflow.
  • GitHub Pages and GitHub Actions - GitHub service terms plus action-specific repository licenses. Source: github.com/actions. Used for static site hosting and deployment.
